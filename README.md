# 🚀 Rembugan

"LinkedIn for Students" — Platform Kolaborasi, Matchmaking Proyek, dan Portofolio.

## 📂 Monorepo Structure
- `rembugan-backend/`: FastAPI API service (Python, Prisma, PostgreSQL).
- `rembugan_frontend/`: Flutter mobile app.
- `rembugan-dashboard/`: Next.js admin dashboard.

## 🛠️ Stack
- **Backend:** Python, FastAPI, Prisma, Gemini AI, Firebase Auth.
- **Frontend:** Flutter, GetX.
- **Dashboard:** Next.js, Shadcn UI, Tailwind CSS.

*(See README inside each folder for specific run instructions).*
